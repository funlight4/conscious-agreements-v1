## Page Summaries

### Foundation Section

**what-is-agreement.html**
Core concepts: What makes an agreement, essential elements (offer, acceptance, meeting of minds, consideration, capacity, lawful purpose). Key insight: most "law" traces back to agreement. Defines consideration as critical test.

**conscious-unconscious.html** [COMPLETE]
The distinction between agreements knowingly entered vs. stumbled into. How unconscious agreements form through assumption, silence, behavior, and misunderstood terms. Comparison table of characteristics. Links to Terminology page for key terms.

**how-you-energize.html** [COMPLETE]
The five ways you keep agreements alive: compliance, identification, silence, using benefits, payment. De-energizing options table. Awareness practice framework.

**covert-contracts.html** [COMPLETE]
Hidden agreements formed through status, presumption, behavior, and definitional traps. One-sidedness problem and consideration test. Recognition signs and response strategies.

**healthy-agreements.html** [COMPLETE]
Seven marks of healthy agreements with test questions for each: clear mutual understanding, genuine consent, valid consideration, full capacity, lawful purpose, clear exit terms, balanced consequences. Checklist for new agreements and evaluation framework for existing ones.
