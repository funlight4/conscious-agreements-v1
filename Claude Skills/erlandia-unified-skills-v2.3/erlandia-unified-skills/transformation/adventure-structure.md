## The Adventure Framework

### Core Components

**1. Rank System (Consciousness Levels)**
- **Seeker** (Level 1-10): Beginning the journey
- **Explorer** (Level 11-25): Discovering the path
- **Practitioner** (Level 26-50): Applying wisdom
- **Adept** (Level 51-75): Mastering fundamentals
- **Master** (Level 76-90): Teaching others
- **Sage** (Level 91-99): Embodying wisdom

**2. Experience Points (XP) Types**
- **Wisdom Points (WP):** Knowledge acquisition and integration
- **Magic Points (MP):** Transformation and breakthroughs
- **Karma Points (KP):** Service and contribution
- **Creation Points (CP):** Content and beauty creation
- **Unity Points (UP):** Community and connection

**3. Achievement Categories**
- **Journey Achievements:** Milestones on paths
- **Skill Achievements:** Mastery demonstrations
- **Service Achievements:** Helping others
- **Creation Achievements:** Contributing content
- **Breakthrough Achievements:** Major transformations
- **Secret Achievements:** Hidden discoveries

**4. Unlock System**
- **Content Unlocks:** New teachings available
- **Tool Unlocks:** Advanced resources accessible
- **Path Unlocks:** New journeys opened
- **Ability Unlocks:** New capabilities earned
- **Community Unlocks:** Access to groups/mentors

### Point Calculation Formulas

**Base XP Formula:**
```
XP Earned = (Proficiency √ó Difficulty √ó Completion) √ó Multipliers

Where:
- Proficiency = 0-1 (skill demonstration)
- Difficulty = 1-10 (challenge level)
- Completion = 0-1 (how much finished)
- Multipliers = Various bonuses
```

**Multiplier Examples:**
- First completion: 2x
- Perfect execution: 1.5x
- Helping others: 1.3x
- Creative approach: 1.4x
- Speed bonus: 1.2x
- Party bonus: 1.1x per member
